# Generate W&B Models CLI

Generate markdown reference documentation for [W&B CLI commands](https://github.com/wandb/wandb/tree/main/wandb/cli).


## Usage

Specify the `wandb` GitHub release tag that you want to generate docs for:

```bash
./generate_cli.sh <release_tag>
```

Example:
```bash
./generate_cli.sh v0.18.3
```

## What it does

1. Clones/updates the `wandb/wandb` repo to `./wandb/`.
2. Checks out the specified release tag.
3. Discovers all public (non-hidden) CLI commands. Stores information to JSON file.
4. Generates markdown docs.
5. Sorts markdown files.

Output is written to `./output/`.


## Files

| File | Description |
|------|-------------|
| `generate_cli.sh` | Main script that orchestrates the pipeline |
| `get_public_commands.py` | Discovers public CLI commands and their source locations |
| `source_info.json` | Contains info of CLI commands and their properties (options, descriptions, source line, etc.). |
| `create_mdx_file.py` | Reads in JSON file generated by `get_public_commands.py` and `cli_doc_templat.py`. Generates MDX files. |
| `sort_markdown.py` | Creates and sorts subcommands of Click groups. |
| `cli_doc_template.py`| Contains string templates that are read in by `create_mdx_file.py` to format the MDX files. |
